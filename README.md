**landin_page-MaratonaTech**

**Descrição:** Projeto simples de landing page criado durante a Maratona Tech. Esta página demonstra um layout estático com HTML, CSS e JavaScript para efeitos e interações básicas.

**Visualização rápida:** Abra o arquivo `index.html` no seu navegador (duplo-clique) ou execute o comando abaixo no Windows:

```cmd
start index.html
```

**Funcionalidades:**
- **Layout responsivo:** design simples preparado para telas comuns.
- **Estilos customizados:** arquivo `css/style.css` para estilos da página.
- **Interações JavaScript:** comportamentos mínimos no `js/main.js`.

**Estrutura do projeto:**
- `index.html` : página principal da landing page.
- `css/style.css` : estilos da página.
- `js/main.js` : scripts e interações.
- `README.md` : este arquivo.

**Tecnologias:**
- HTML5
- CSS3
- JavaScript (vanilla)

**Como executar localmente**
- Método simples (modo gráfico): dê um duplo-clique em `index.html` para abrir no navegador.
- Via linha de comando (Windows - `cmd`):

```cmd
cd "c:\Users\Victor Alves\Documents\GitHub\meu_portifolio\landin_page-MaratonaTech"
start index.html
```

Opcional: se quiser servir via servidor local (útil para caminhos relativos e CORS), instale um servidor simples como `http-server` (Node.js) e rode:

```cmd
npm install -g http-server
http-server -c-1

# depois abra o endereço mostrado no terminal, por exemplo http://localhost:8080
```

**Como personalizar**
- Atualize textos e seções em `index.html`.
- Altere cores, fontes e layout em `css/style.css`.
- Adicione ou modifique scripts em `js/main.js`.
